# 個人成長記録ツール

## このツールを作った背景

社会人として日報を書く中で、「記録を会社に提出するだけで終わっている」という課題を感じていた。
自分の成長・感情・気づきを**自分自身のために**蓄積・分析するツールが欲しいと思い開発した。

## 何ができるか

- その日やったこと・学んだこと・感情・明日の予定を入力する
- Claude APIが過去の記録をもとに成長の気づきとアドバイスをフィードバックしてくれる
- 記録がJSONファイルに蓄積され、継続するほど精度が上がる仕組み

## 使い方

### 1. 必要なライブラリをインストール

pip install anthropic python-dotenv


### 2. APIキーを設定

.envファイルを作成して以下を記述：

API_KEY=自分のClaude APIキー


### 3. 実行

python nikki.py


## 今後追加したい機能

- 過去の記録を一覧表示する機能
- 週次・月次での成長サマリーを生成する機能
- 感情の変化をグラフで可視化する機能

## 使用技術

- Python 3.14
- Claude API (Anthropic)
- python-dotenv
